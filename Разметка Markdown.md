# 3. Списки и отступы

## 3.1 Маркированные списки

```
* Пункт 1
    * Пункт 1.1
        * Пункт 1.1.1
- Пункт 2
    - Пункт 2.1
        - Пункт 2.1.1
+ Пункт 3
    + Пункт 3.1
        + Пункт 3.1.1
```

* Пункт 1
    * Пункт 1.1
        * Пункт 1.1.1

    - Пункт 2.1
        - Пункт 2.1.1
+ Пункт 3

        + Пункт 3.1.1

## 3.2 Нумерованные списки

```
1. Пункт 1
    1. Пункт 1.1
        1. Пункт 1.1.1

3. Пункт 3
```
